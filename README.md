# CarrIA – AI-Powered Career Guidance Platform

CarrIA is an interactive platform that uses artificial intelligence and behavioral assessments to help young people discover their professional profiles, develop skills, and find opportunities aligned with their potential.

---

## Features

- Interactive behavioral profile assessment  
- Multilingual support (Portuguese, English, Spanish)  
- Gamified experience with quizzes and games  
- AI-powered personalized recommendations  
- Profile analysis (Strategic, Creative, Execution, Relational)  
- Modern and responsive interface  

---

## Tech Stack

- Next.js  
- React  
- TypeScript  
- Tailwind CSS  
- Vercel (deployment)  
- Internationalization system (i18n)
